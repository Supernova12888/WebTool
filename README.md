# WebTool

## About
Using DOM in JavaScript can be pretty frustrating, confusing, and hard to use. But WebTool fixes that problem! With WebTool, you will be able to type less and help you get your website finished faster.

## Installation
To install WebTool, just insert this piece of code into your HTML code. The `[VERSION]` part should be replaced with the version of WebTool you want to install. You may also replace `[VERSION]` with `latest` to get the latest version of WebTool
```html
<script src="https://cdn.jsdelivr.net/gh/Supernova12888/WebTool@[VERSION]/index.js"></script>
```
